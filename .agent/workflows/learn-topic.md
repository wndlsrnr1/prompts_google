# 1. Deconstruct the Subject
Break the topic into its most basic, non-reducible elements.
- Tool: `view_file .agent/skills/learning/deconstructing-knowledge/SKILL.md`
- Action: List the fundamental atoms of the topic.

# 2. Map the Learning Path
Organize the atoms into a logical sequence from Foundation to Mastery.
- Tool: `view_file .agent/skills/learning/mapping-curriculum/SKILL.md`
- Action: Create a "Base to Top" roadmap with clear milestones.

# 3. Deep Dive (Optional)
For any specific atom, perform targeted research or implementation.
- Tool: `view_file .agent/skills/common/planning/SKILL.md`

# 4. Verify Mastery
Test the understanding of the current layer before moving up.
- Tool: `view_file .agent/skills/learning/testing-mastery/SKILL.md`

- Action: Conduct a Feynman technique check or active recall quiz.
